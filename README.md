# Hand Cricket Game

A simple browser-based Hand Cricket game built with modern frontend tooling.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Notes on Pushing to GitHub

To push commits from a local environment or automation agent, GitHub authentication is required (typically a Personal Access Token or SSH key). Without valid credentials, a `git push` will fail even if commits are created successfully.
